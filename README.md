# Valeria Batiz — Presentación Personal

Sitio web personal con CV interactivo.

## 🚀 Cómo subir a GitHub Pages

1. Creá un repositorio en GitHub (ej: `valeria-cv`)
2. Subí el archivo `index.html` al repositorio
3. Andá a **Settings → Pages → Branch: main → Save**
4. En 1-2 minutos tu sitio estará en: `https://TU-USUARIO.github.io/valeria-cv/`

## 🔑 Clave para ver el CV
La clave para desbloquear el CV es: **Valeria**
